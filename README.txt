===== FunPay OTC Mini App v2.0 =====

НОВОЕ В v2.0:
- Сделки синхронизируются с сервером (обе стороны видят актуальный статус)
- Реальная система эскроу через менеджера @FunPay_Safety
- Автообновление статуса каждые 5 секунд
- Диалоги подтверждения для критических действий

КАК РАБОТАЕТ ЭСКРОУ (точно по шагам):
1. Продавец создаёт сделку → делится ссылкой
2. Покупатель присоединяется → оплачивает с баланса
3. Продавец видит "Оплачена" → передаёт товар менеджеру @FunPay_Safety
4. Продавец нажимает "Передал менеджеру" (с подтверждением)
5. Покупатель связывается с @FunPay_Safety, проверяет товар
6. Покупатель нажимает "Подтвердить получение" → деньги к продавцу

НАСТРОЙКА API (для онлайн-сделок):
1. Запустите бот-сервер
2. Узнайте его публичный URL
3. В index.html найдите: window.FUNPAY_API_BASE = ''
4. Замените на: window.FUNPAY_API_BASE = 'https://your-bot.railway.app'
5. Загрузите на GitHub Pages

КАК ЗАГРУЗИТЬ НА GITHUB PAGES:
1. Репозиторий: funpay-otc
2. Загрузите все 4 файла (index.html, style.css, app.js, tonconnect-manifest.json)
3. Settings → Pages → Source: main / root
4. URL: https://ВАШ_ЮЗЕ.github.io/funpay-otc

В @BotFather: /mybots → Bot Settings → Menu Button → URL мини-аппа
