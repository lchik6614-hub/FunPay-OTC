===== FunPay OTC Mini App =====

КАК ЗАГРУЗИТЬ НА GITHUB PAGES:
1. Создайте репозиторий: funpay-otc
2. Загрузите все файлы из этой папки (index.html, style.css, app.js)
3. Settings → Pages → Source: main branch / root
4. URL будет: https://ВАШ_ЮЗЕР.github.io/funpay-otc

ЗАТЕМ:
- Скопируйте этот URL в .env бота (MINI_APP_URL=...)
- Перезапустите бота

НАСТРОЙКА В @BotFather:
/mybots → ваш бот → Bot Settings → Menu Button → Set URL → вставьте URL мини-аппа

ВАЖНО: Сначала загрузите мини-апп на GitHub Pages, потом настройте бота!
